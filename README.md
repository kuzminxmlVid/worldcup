# World Cup Bot — PNG flags + autoreminders version

Что добавлено:

- PNG-карточка следующего матча с флагами
- автопост за 1 час до матча
- автопост можно включать и выключать
- кнопка `Автопост`
- команды:
  - `/alerts` — переключить автопост
  - `/alerts_on` — включить
  - `/alerts_off` — выключить
- inline-кнопка под сообщениями показывает текущий статус автопоста

## После обновления

1. Замени файлы в GitHub
2. Сделай redeploy
3. В Telegram нажми `/start`
4. Проверь:
   - `/next`
   - кнопку `Следующий матч`
   - кнопку `Автопост`
   - `/alerts_on`
   - `/alerts_off`

По умолчанию автопост включён.
