# World Cup Bot — FlagCDN HD flat flags

Что изменено:

- убран кривой эффект развевающегося флага
- карточка использует обычные ровные флаги
- бот сам скачивает PNG-флаги с FlagCDN при первом создании карточки
- скачанные флаги кэшируются в `/tmp/worldcup_flags_hd`
- если сеть недоступна, бот использует локальный fallback из `assets/flags`
- автопост за 1 час до матча сохранён
- включение/выключение автопоста сохранено

## Важно

На первом `/next` бот может рисовать карточку чуть дольше, потому что скачивает флаги.
Потом флаги берутся из кэша.

## После деплоя

Проверь:

```text
/start
/next
/alerts
```
