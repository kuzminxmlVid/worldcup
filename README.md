# World Cup Bot — illustrated UX version

Что нового:

- карточка-иллюстрация для команды «Следующий матч» и команды `/next`
- улучшенный UX
- нижняя клавиатура: Сегодня, Завтра, Следующий матч, 7 дней, Обновить, Меню
- inline-кнопки под сообщениями
- более короткий и понятный стартовый экран
- локальный `schedule.json`, без внешних API

## Что заменить в GitHub

Замени весь набор файлов из архива в корне репозитория.

Новые важные файлы:
- `keyboards.py`
- `match_card.py`

Обновлённые:
- `main.py`
- `formatting.py`
- `requirements.txt`

## После деплоя

1. Сделай redeploy
2. В Telegram нажми `/start`
3. Проверь:
   - `Следующий матч`
   - `/next`
   - `Сегодня`
   - `7 дней`
