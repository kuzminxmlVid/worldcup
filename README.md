# World Cup Bot — local PNG flag assets + autoreminders

Что исправлено:

- флаги в PNG-карточке больше не зависят от emoji-шрифтов Railway
- в проект добавлена папка `assets/flags`
- карточка следующего матча берёт флаги из локальных PNG-файлов
- автопост за час до матча можно включать и выключать

## Команды автопоста

- `/alerts` — переключить
- `/alerts_on` — включить
- `/alerts_off` — выключить

## После обновления

1. Замени файлы в GitHub целиком.
2. Обязательно залей папку `assets/flags`.
3. Сделай redeploy.
4. Нажми `/start`.
5. Проверь `/next`.

Если флаги снова не видны, значит папка `assets/flags` не попала в GitHub.
