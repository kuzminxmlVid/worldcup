# World Cup Bot — predictions and notes

Добавлены две фишки:

- личный прогноз пользователя на матч
- личная заметка пользователя по матчу

Данные хранятся в PostgreSQL в таблице `user_match_data`.

## Как это работает

1. Пользователь нажимает `Следующий матч` или `/next`
2. Бот показывает карточку матча
3. Под матчем появляются кнопки:
   - `Сделать прогноз`
   - `Добавить заметку`
   - `Показать мои данные`
   - `Очистить`

Прогноз и заметка хранятся отдельно для каждого пользователя.

## Ограничения

- прогноз: до 200 символов
- заметка: до 2000 символов

Если заметка длиннее 2000 символов, бот не сохранит её и попросит сократить.

## После обновления

1. Замени файлы в GitHub
2. Сделай redeploy в Railway
3. Нажми `/start`
4. Проверь:
   - `/next`
   - `Сделать прогноз`
   - `Добавить заметку`
   - `Показать мои данные`
