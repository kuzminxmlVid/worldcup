# World Cup Bot — next match version

Что добавлено:

- кнопка «Следующий матч»
- команда `/next`
- следующий матч отправляется отдельным сообщением
- флаги рядом со сборными
- группировка матчей по датам
- кнопки: Сегодня, Завтра, Следующий матч, 7 дней, Обновить, Источник, Debug

## После обновления

1. Замени файлы в GitHub.
2. Сделай redeploy на Railway.
3. В Telegram нажми `/start`, чтобы обновилась клавиатура.

## Проверка

```text
/source
/sync
/debug
/next
```

Или нажми кнопку «Следующий матч».
